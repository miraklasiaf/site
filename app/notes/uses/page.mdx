### Computer / Laptop

- Lenovo Ideapad Slim 3 Ryzen 3 4300U

### Coding Tools

- Visual Studio Code (Frontend) ([Extensions](https://gist.github.com/miraklasiaf/2cb553fe9b41b66658d225eb43189468))
- Visual Studio (Backend)
- MySQL Workbench
- Fiddler
- Postman

### Tech Stack

- Framework: [React](https://reactjs.org/) / [Next.js](https://nextjs.org/)
- UI Library: [MUI](https://mui.com/) and [Tailwind CSS](https://tailwindcss.com/)
- API: ASP.NET Web API 2
- Database: MySQL
